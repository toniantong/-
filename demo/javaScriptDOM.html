、一個按鈕或一段文字。 2. Document：網頁的統治者 document 物件提供了許多方法，讓我們可以在網頁中**尋找**現有的元素，或是**建立**新的元素。 A. 尋找元素 (Selecting Elements) 最常用的兩個方法是 getElementById 和 querySelector。 // 透過 ID 尋找單一元素 const myTitle = document.getElementById('title-id'); // 透過 CSS 選擇器尋找 (尋找第一個符合的) const myBox = document.querySelector('.box-class'); // 尋找所有符合條件的元素 (回傳 NodeList) const allButtons = document.querySelectorAll('button'); B. 建立新元素 (Creating Elements) 使用 createElement 可以在記憶體中做出一個新的 HTML 標籤，然後用 appendChild 把它放進網頁中。 互動 Demo：建立元素 點擊建立新段落 const btn = document.getElementById('demo1-btn'); const container = document.getElementById('demo1-container'); btn.addEventListener('click', () => { // 1. 建立一個新的 元素 const newElement = document.createElement('p'); // 2. 設定內容與樣式 newElement.textContent = '這是一段用 JavaScript 動態建立的文字！ 🎉'; newElement.className = 'bg-green-100 text-green-800 p-2 rounded'; // 3. 將元素加入到網頁的容器中 container.appendChild(newElement); }); 3. Element：操作單一標籤 一旦你透過 document 找到了某個 element（元素），你就可以開始改變它的長相、內容或屬性。 常用屬性與方法 element.textContent：修改或讀取元素的純文字內容。 element.innerHTML：修改或讀取元素的 HTML 結構。 element.style：直接修改 CSS 樣式（行內樣式）。 element.classList：新增、移除或切換 CSS 類別 (Class)。 互動 Demo：修改元素的樣式與內容 我是一個普通的方塊 改變顏色 改變文字 切換形狀 const box = document.getElementById('demo2-box'); const textSpan = document.getElementById('demo2-text'); // 改變顏色 (修改 classList) document.getElementById('demo2-btn-color').addEventListener('click', () => { box.classList.toggle('bg-blue-500'); box.classList.toggle('bg-red-500'); }); // 改變文字 (修改 textContent) document.getElementById('demo2-btn-text').addEventListener('click', () => { textSpan.textContent = textSpan.textContent === '被修改了！' ? '我是一個普通的方塊' : '被修改了！'; }); // 切換形狀 (修改 classList) document.getElementById('demo2-btn-shape').addEventListener('click', () => { box.classList.toggle('rounded-lg'); box.classList.toggle('rounded-full'); }); 4. 事件 (Events)：讓元素活起來 前面示範中一直在使用的 addEventListener 就是用來監聽使用者在 element 上做的動作（例如點擊、滑鼠移動、鍵盤輸入等）。 互動 Demo：輸入框事件 請輸入任意文字： 你目前輸入的內容： const inputEl = document.getElementById('demo3-input'); const outputEl = document.getElementById('demo3-output');
